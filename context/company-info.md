# 다이버즈 도메인 컨텍스트

## 기업 소개

**다이버즈(Divers)** 는 60만 간호사·간호학생 커리어 플랫폼 **널스빌리지(NurseVillage)** 와 병의원 채용관리 SaaS **메디하이어(MediHire)** 를 운영하는 헬스케어 HR 스타트업입니다.

- **미션**: "사람으로부터 시작되는 더 나은 의료서비스"
- **핵심 전략**: 200만 보건의료인 HR 시장의 디지털 전환 — 구인·구직, 커뮤니티, 채용관리를 한 데이터 위에 연결
- **주요 성과**:
  - 매쉬업벤처스·신용보증기금으로부터 투자 유치
  - 경희대학교·동국대학교 캠퍼스타운 사업 선정
  - 널스빌리지 누적 회원 60만 명(간호사·간호학생) 확보
- **운영 서비스**: 널스빌리지(B2C 커뮤니티·커리어), 메디하이어(B2B 병의원 채용관리)

## 주요 서비스

### 널스빌리지 (NurseVillage)

간호사·간호학생을 위한 커리어·커뮤니티 플랫폼

| 기능 | 설명 |
|------|------|
| 커뮤니티 | 교대근무·태움·이직 고민 등 간호 현장 이야기를 나누는 공간 |
| 채용·이직 | 병동·근무형태별 채용 정보, 경력 RN·신규 RN 맞춤 매칭 |
| 콘텐츠·뉴스레터 | 임상 정보·취업 가이드·국시 정보를 카드뉴스·카카오·뉴스레터로 발행 |
| 국시·실습 정보 | 간호학생 대상 국가시험·실습·면허 정보 제공 |

### 메디하이어 (MediHire)

병의원 채용관리 SaaS — 채용 공고·지원자 관리·면접 일정을 한 곳에서 운영

## 콘텐츠 채널 포맷

널스빌리지는 한 가지 주제를 **3종 포맷**으로 동시 발행합니다.

| 채널 | 분량·구성 | 특성 |
|------|-----------|------|
| 카드뉴스 | 3~5장, 장별 헤드라인 + 본문 2~3문장, 마지막 장 CTA | 스토리텔링·스와이프, 인스타·피드 노출 |
| 카카오메시지 | 200자 이내, 핵심 + 링크 유도 | 짧고 강한 훅, 즉시 클릭 유도 |
| 뉴스레터 | 600자 내외, 도입–본문–CTA 3단 | 정보 깊이·신뢰, 정기 구독자 대상 |

## 고객 세그먼트

| 세그먼트 | 특징 | 주요 니즈 |
|----------|------|-----------|
| 신규 RN (신규간호사) | 입사 1년 이하, 프리셉터십·적응기 | 현장 적응 공감, 투약·인계 실무 팁, 번아웃 위로 |
| 간호학생 | 재학·실습·국시 준비 | 국가시험·실습·면허 정보, 임상 진입 준비 |
| 경력 RN (경력간호사) | 3년차 이상, 이직·재취업 검토 | 이직 정보, 처우·연봉, 간호법 등 제도 변화 |

## 이 과제가 중요한 이유 (비즈니스 맥락)

널스빌리지의 경쟁력은 **간호 현장에 깊이 공감하는 콘텐츠**입니다. 일반 카피라이터가 쓰면 현장감이 떨어져 공감도가 낮아지고, 60만 회원의 신뢰가 약해집니다.

**콘텐츠 3종 자동 생성이 중요한 이유:**

1. **생산 속도**: 1건 2~3시간 × 3종 → AI 활용 시 5분 내 초안. 마케터 1명의 병목 해소
2. **브랜드 보이스 일관성**: 담당자가 바뀌어도 톤·현장 용어 반영이 흔들리지 않음
3. **시의성 대응**: 국시 합격발표·간호법 시행 같은 이벤트에 빠르게 콘텐츠로 대응
4. **채널 최적화**: 같은 주제도 카드/카카오/뉴스레터 특성에 맞게 변형해 도달·전환 향상
5. **공감 기반 리텐션**: 현장 용어가 살아있는 콘텐츠가 커뮤니티 체류·재방문을 끌어올림

## 간호 현장 용어 사전

| 용어 | 정의 |
|------|------|
| **나이팅게일 선서** | 간호학생이 임상 실습에 들어가기 전 거행하는 선서식. 간호 윤리·소명을 다짐하는 상징적 의례. |
| **태움** | "재가 될 때까지 태운다"에서 유래한 간호계 직장 내 괴롭힘 관행을 가리키는 현장 용어. 신규 RN 적응의 대표적 고통점. |
| **RN** | Registered Nurse. 간호사 국가시험에 합격해 면허를 받은 정식 간호사. |
| **LPN** | Licensed Practical Nurse(국내 통용상 간호조무사 등 실무 인력). RN과 업무 범위·면허가 구분됨. |
| **프리셉터(Preceptor)** | 신규 간호사를 1:1로 교육·지도하는 선배 간호사. 프리셉터십 = 신규 적응 교육 제도. |
| **SBAR** | Situation-Background-Assessment-Recommendation. 환자 상태를 구조화해 인계·보고하는 표준 의사소통 기법. |
| **인계(인수인계)** | 듀티 교대 시 환자 상태·처치 내용을 다음 근무자에게 전달하는 것. |
| **듀티표** | 데이(D)·이브닝(E)·나이트(N) 근무를 배정한 교대 근무표. |
| **데이/이브닝/나이트** | 3교대 근무 구분. 데이(오전)·이브닝(오후)·나이트(야간). |
| **3교대** | 데이·이브닝·나이트로 24시간을 나누는 간호사 교대근무 형태. |
| **라운딩(Rounding)** | 회진. 환자 병실을 돌며 상태를 확인하는 활동. |
| **바이탈(Vital Sign)** | 활력징후(혈압·맥박·호흡·체온 등) 측정·기록. |
| **콜벨** | 환자가 간호사를 호출하는 벨. 병동 업무 강도의 상징적 표현으로도 쓰임. |
| **5R(Five Rights)** | 투약 안전 원칙 — 정확한 대상자·약물·용량·경로·시간. 투약 오류 예방의 기본. |
| **OCS** | Order Communication System. 처방전달시스템. 의사의 처방을 전산으로 전달·확인. |
| **EMR** | Electronic Medical Record. 전자의무기록. 환자 진료·간호 기록을 전산화한 시스템. |
| **국시(국가시험)** | 간호사 국가시험. 합격 후 RN 면허 취득. 매년 초 시행·합격발표. |
| **간호법** | 간호사의 업무 범위·처우 등을 규정하는 법률. 시행 시 현장 업무·처우에 변화. |
| **번아웃** | 교대근무·고강도 업무로 인한 신체적·정서적 소진 상태. |
| **더블듀티** | 연속·중복 근무. 인력 부족 시 발생하는 과중 근무. |
