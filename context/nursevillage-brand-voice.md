# 널스빌리지 브랜드보이스 가이드

> 2026-07-01, output_content_set.md 18건 전체 정합화 작업에서 확정한 표현 규칙.
> 이후 신규 세트를 생성할 때도 아래 3가지 기준을 기본값으로 적용하세요.

## 1. 호칭 — "선생님" + 기능적 지칭 병기

"신규 RN", "간호학생", "경력RN" 같은 기능적 지칭(용어사전·keywords_required와 매칭되는 값)은
그대로 유지하고, 그 옆에 "선생님" 존칭을 붙여 병기합니다.

- 신규RN → **"신규 RN 선생님"**
- 간호학생 → **"간호학생 선생님"**
- 경력RN → **"경력RN 선생님"**

### 배치 원칙
- 세트(카드뉴스+카카오+뉴스레터) 전체에서 **최소 1회** 등장하면 충족. 매 문장 반복하지 않습니다.
- 가장 자연스러운 지점 하나를 골라 넣습니다 — 보통 카드뉴스 CTA(마지막 장)나 뉴스레터 도입부처럼
  독자에게 직접 말을 거는 문장이 적합합니다.
- 이미 "선생님"이 다른 의미(프리셉터·실습 지도 선생님 등 인물 지칭)로 쓰인 문장은 그대로 두고,
  별도로 호칭용 "선생님"을 추가합니다. 두 용법이 한 세트에 같이 있어도 무방합니다.
  (예: 시드#08 "실습생을 맡아주는 선생님" = 지도자 지칭, "간호학생 선생님" = 독자 호칭 — 서로 다른 용도)

## 2. 감성 이모지

문장 끝에 감성 이모지를 자연스럽게 섞습니다. 아이콘성 이모지(✅📋🩺 등)가 아니라
감정을 전달하는 이모지를 씁니다.

- 팔레트: 🤍 😊 🥲 💪 🌱 😆 🙈 😅 (필요시 상황에 맞는 다른 감성 이모지 추가 가능, 예: 축하 맥락의 🎉)
- 톤별 가이드:
  - 공감형 → 🤍 😊 🥲
  - 정보형 → 😊 💪
  - 유머형 → 😆 🙈 😅
- 배치량: **카드뉴스 4장 중 1~2장**, **뉴스레터 1~2곳**(보통 도입부 끝·마무리 CTA 끝)에만 사용.
  카카오메시지는 짧고 즉시 클릭을 유도하는 채널 특성상 이모지를 넣지 않습니다.
- 한 문장에 이모지 1개만 — 과하게 겹쳐 쓰지 않습니다.

## 3. 증언 인용 포맷 — 사용하지 않음

"OO병원 n년차" 같은 구체적 인물·소속·연차를 특정하는 증언 포맷은 **쓰지 않습니다.**
data/nursevillage_content_seeds_cleaned.csv에는 실제 인터뷰·증언 데이터가 없어서,
이런 형식을 쓰면 존재하지 않는 사람의 말을 지어내는 가짜 인용이 됩니다.

대신 이미 일부 콘텐츠에 쓰였던 **익명 집단 화법**을 18건 전체에 일관되게 적용합니다.

- 표준 표현: "선배들의 이야기를 들어보면, ~라고 말합니다" / "~들은 압니다" / "다들 ~라고 합니다"
- 특정 개인을 지시하는 대명사·소속·연차를 넣지 않고, 항상 집단("선배들", "다들", "~한 사람들")으로 뭉뚱그려 표현
- 1인칭 회고 시점 콘텐츠(예: "3교대 10년차가 후배에게 전하는 위로")는 화자 스스로가 곧 선배이므로
  위 문형을 억지로 추가하지 않아도 됩니다 — 이미 화자 본인의 경험담이라 별도 증언 인용이 필요 없습니다.

### 한계 (알려진 제약)
증언 인용은 **이번 세트에는 적용하지 않았습니다.** 근거 데이터(실제 간호사 인터뷰·후기)가
없는 상태에서 형식만 갖추면 가짜 인용이 되기 때문입니다. **실제 증언·인터뷰 데이터를 확보하면**,
익명 집단 화법 대신 (동의를 받은) 실명·소속 인용으로 교체하는 것을 권장합니다.

## 글자수 재검증 원칙

호칭·이모지를 추가하면 분량이 늘어나므로, 수정 후 항상 아래를 다시 실측합니다.

- 카카오메시지: 200자 이내 (이모지는 넣지 않으므로 보통 영향 적음)
- 뉴스레터: 550~650자 권장 범위 — 도입부 1문단 + 본문 1문단 + CTA 1문단을 `\n\n`으로 이어붙인
  전체 글자수(공백 포함)로 계산. 범위를 벗어나면 문장을 늘리거나 줄여 재조정하고,
  헤더의 "(n자)" 표기도 실제 값으로 다시 갱신합니다.
