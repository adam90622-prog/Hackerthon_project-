## 6. 재현성

```
python scripts/reschedule.py --today 2026-07-01 --horizon-end 2026-12-17 --committed-days 14 --insert-lead-days 3
```

동일한 published_log.csv·classified_new_articles.csv가 입력되면 항상 동일한
재스케줄링 결과가 재현됩니다(라운드로빈 순서 = 미사용 아티클 id 오름차순 →
과거 발행 이력이 오래된 순, 70일 쿨다운 미충족 시 다음 후보로 자동 스킵,
확정 구간 삽입은 리드타임 → 슬롯 순으로 결정론적으로 판정).
