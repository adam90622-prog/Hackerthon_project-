# [04] 다이버즈 — SNS 콘텐츠 3종 동시 생성

트랙: 마케팅 / PM | 작업 시간: 3시간 | 개인 과제
채용 TO: 콘텐츠 마케터 1명

## 기업 배경

다이버즈(Divers)는 60만 간호사·간호학생의 커리어 플랫폼 **널스빌리지**와 병의원 채용관리 SaaS **메디하이어**를 운영하는 헬스케어 HR 스타트업입니다. "사람으로부터 시작되는 더 나은 의료서비스"를 미션으로, 200만 보건의료인 HR 시장의 디지털 전환을 추진합니다. 매쉬업벤처스·신용보증기금의 투자를 유치했고, 경희대·동국대 캠퍼스타운 사업에 선정된 성장형 스타트업입니다.

## 현재 상황과 문제

널스빌리지는 간호사·간호학생을 대상으로 SNS 콘텐츠(카드뉴스·카카오메시지·뉴스레터)를 정기 발행합니다. 그러나 콘텐츠 마케터 1명이 다음 어려움을 안고 있습니다.

1. **3종 포맷 동시 제작 부담**: 하나의 주제로 카드뉴스·카카오메시지·뉴스레터를 각각 작성 → 1건당 2~3시간 소요
2. **현장 용어 누락 → 공감도 하락**: 간호 현장 용어(나이팅게일 선서·태움·RN/LPN·프리셉터)가 빠지면 독자 공감도가 낮아짐
3. **브랜드 보이스 흔들림**: 담당자가 바뀌면 톤이 흔들려 일관성이 깨짐
4. **시의성 대응 느림**: 기획→초안→수정 사이클이 길어 국시 합격발표·간호법 같은 시의성 이벤트 대응이 늦음

## 미션

data/nursevillage_content_seeds.csv 파일을 분석하여, 각 행(주제·톤·타깃)마다:
1. **카드뉴스 텍스트** 3~5장 — 장별 헤드라인 + 본문 2~3문장, 마지막 장에 CTA
2. **카카오메시지** 200자 이내 — 핵심 메시지 + 링크 유도
3. **뉴스레터** 600자 내외 — 도입–본문–CTA 3단 구성
을 한 번에 생성하고, 각 행에서 **사용된 현장 용어 목록**을 태그 형식으로 정리하세요.

공통 필수: keywords_required의 간호 현장 용어를 모든 채널에 자연스럽게 반영해야 합니다.

## 입력 데이터

data/nursevillage_content_seeds.csv — 18건의 콘텐츠 시드 데이터
구성: target_audience 3종(신규RN·간호학생·경력RN) × tone 3종(공감형·정보형·유머형), 각 조합 2건씩 = 18건.

| 컬럼 | 설명 |
|------|------|
| id | 시드 일련번호 (01~18) |
| topic | 콘텐츠 주제 |
| tone | 톤 (공감형 / 정보형 / 유머형) |
| target_audience | 대상 독자 (신규RN / 간호학생 / 경력RN) |
| keywords_required | 반드시 포함할 현장 용어 (예: "프리셉터,태움,SBAR") |
| platform_hint | 우선 노출 채널 힌트 (kakao / card / newsletter) |
| ref_event | 연계 시의성 이벤트 (예: "간호사 국가시험 합격발표") |

데이터 품질 주의사항 (정제가 필요합니다):
- **keywords_required 결측 1건**: id 12는 현장 용어가 비어 있음 → topic·target·tone 기반으로 적절한 현장 용어를 추론해 보완 필요
- **ref_event 빈칸 다수**: 시의성 이벤트가 없는 상시 콘텐츠 → 빈칸은 시즌·이벤트 언급 없이 작성
- **중복 topic 1건**: id 11과 id 17은 topic 문자열이 동일("실습 나가서 듣는 웃픈 한마디 모음")하나 target_audience가 다름(간호학생 vs 경력RN) → 같은 제목이라도 타깃별로 다르게 차별화해야 함
