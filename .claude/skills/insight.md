# /insight — 채널 전략 + 현장 용어 활용 계획

## 목적

/analyze 결과를 바탕으로 각 시드를 카드뉴스·카카오메시지·뉴스레터 3종으로 변형하는 전략을 세우고,
타깃·톤·시의성·현장 용어 활용 계획을 수립합니다.

---

## 전제 조건
1. /analyze 완료 — 결측·중복·분포 파악됨
2. `context/company-info.md`(용어 사전·세그먼트) 읽기 완료
3. `context/industry-news.md`(시의성 이벤트·채널 트렌드) 읽기 완료

---

## 전략 수립 항목

### 1. 타깃별 톤 매핑
- **신규RN**: 적응기 공감·실무 팁. 공감형은 위로, 정보형은 안전·실무, 유머형은 흑역사 공감.
- **간호학생**: 국시·실습·면허 정보. 미래 RN 관점의 준비·기대 톤.
- **경력RN**: 이직·처우·제도(간호법). 신뢰·현실 조언 톤.

### 2. 채널별 변형 규칙
| 채널 | 변형 방향 |
|---|---|
| 카드뉴스 | 톤을 스토리텔링으로 — 장별로 호흡 나누고 마지막 장 CTA |
| 카카오메시지 | 톤을 한 문장 훅 + 링크 유도로 압축 (200자 이내) |
| 뉴스레터 | 톤을 도입–본문–CTA 3단의 정보 깊이로 확장 (600자 내외) |

### 3. 현장 용어 활용 계획
- keywords_required의 용어를 **전 채널에 최소 1회 이상 자연 삽입**
- 결측(id 12)은 topic·target 기반으로 적절한 용어를 추론해 보완 → decisions.md 기록
- 사용한 용어는 출력 시 **태그 형식**(예: `#프리셉터 #태움 #SBAR`)으로 정리

### 4. 시의성(ref_event) 반영 계획
- ref_event 있는 행: 도입·CTA에 해당 이벤트(국시 합격발표·간호법 등) 자연 연계
- ref_event 빈칸: 상시 콘텐츠로 작성, 특정 시즌 언급 금지

### 5. 중복 topic 차별화 (id 11·17)
- 같은 제목이라도 target이 다르므로(간호학생 vs 경력RN) 사례·어휘·CTA를 달리 설계

```
## 콘텐츠 전략

### 타깃·톤 매핑 요약
### 채널별 변형 규칙
### 현장 용어 활용 계획 (결측 보완 포함)
### 시의성 반영 계획
### 중복 topic 차별화 방향
```
