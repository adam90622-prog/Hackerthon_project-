# /review — 제출 전 자가 채점

## 목적
생성한 3종 콘텐츠 세트가 채점 기준을 충족하는지 제출 전에 점검합니다.

---

## 점검 체크리스트

### 🟢 Basic (100점) 자가 점검
- [ ] **3종 포맷 정확도 (40점)**: 카드뉴스 3~5장(장별 헤드라인+본문, 마지막 장 CTA)·카카오 200자 이내·뉴스레터 600자 내외(도입-본문-CTA) 형식·분량 충족
- [ ] **현장 용어 반영 (20점)**: keywords_required 용어가 전 채널에 자연 삽입 + 사용된 용어 태그 목록 제시
- [ ] **타깃·톤 일치도 (30점)**: target·tone이 채널별로 일관되게 드러남 (단순 복붙 아님)
- [ ] **제출 형식 (10점)**: output_content_set.md 또는 공개 링크

### 글자수 실측 점검
- [ ] 모든 카카오메시지 200자 이내인가? (공백 포함 직접 세기)
- [ ] 모든 뉴스레터 600자 내외인가? (550~650 권장)
- [ ] 카드뉴스 장 수가 3~5장이며 마지막 장이 CTA인가?

### 데이터 처리 점검
- [ ] keywords_required 결측(id 12) 보완값이 표시·기록되었는가?
- [ ] ref_event 빈칸 행은 시즌 언급 없이 작성되었는가?
- [ ] 중복 topic(id 11·17)이 타깃별로 차별화되었는가?
- [ ] 18행 전부 생성되었는가?

### 🟡 Standard (+30점) 점검 (선택)
- [ ] 새 seeds CSV로 재현 가능한 파이프라인(/명령·스크립트)이 있는가?
- [ ] 콘텐츠 캘린더·지표·브랜드보이스 가이드를 본인이 설계했는가? (근거 설명 포함)

### 🔴 Challenge (+30점) 점검 (선택)
- [ ] 발행 전략안 1~2건 (문제 정의·근거 데이터·전략 제안·우선순위)이 있는가?
- [ ] 전략안 양식·우선순위 기준을 본인이 설계했는가? (근거 설명 포함)

```
- Challenge 가산: __ / 30
- 예상 총점: __

### 미흡 항목·보완 계획
- …
```

- 미흡 항목에 보완 계획이 적힘
